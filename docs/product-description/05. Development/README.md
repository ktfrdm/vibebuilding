# 05. Development

Папка для материалов разработки бота (конфигурация, токены, технические выжимки).

## Документы

- **BOT_DEV_SUMMARY.md** — сводка для разработки: сущности, действия бота, интеграция LLM, алгоритмы (из [BOT_FLOW_MVP](../03.%20Scenarios/BOT_FLOW_MVP.md)).
- **IMPLEMENTATION.md** — документация текущей реализации: структура проекта, хранилище, FSM, парсинг дат, запуск.
- **PLAN_IMPLEMENTATION.md** — план реализации бота: User Flow, этапы, порядок задач.

## Токен бота

Токен Telegram-бота хранится в файле **`.env`** (не коммитится в репозиторий).

- Скопируйте `.env.example` в `.env` и подставьте токен от [@BotFather](https://t.me/BotFather).
- В коде загружайте переменные из `.env` (например, через `python-dotenv` или аналог).

**Не добавляйте `.env` в git** — он уже в `.gitignore`.
