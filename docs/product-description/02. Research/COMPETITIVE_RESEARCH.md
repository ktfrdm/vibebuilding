# Исследование конкурентных решений

Способы, которыми люди закрывают потребность договориться о встрече (личные встречи, друзья, небольшие компании). Краткий обзор плюсов, минусов и — где применимо — метрик сервисов.

---

## 1. Личная переписка в мессенджере (чат)

**Суть:** Обмен сообщениями в Telegram, WhatsApp, VK и т.п.: «когда можем?», «я в субботу свободен», «давай в 15:00».

| Плюсы | Минусы |
|-------|--------|
| Привычно, не нужно осваивать новый инструмент | Затягивается на дни, теряется в потоке сообщений |
| Гибко: можно обсудить место, нюансы | Кто-то забывает ответить, приходится напоминать |
| Не требует регистрации в сторонних сервисах | Неясная картина: кто свободен, когда пересечение |
| | Организатор вынужден собирать ответы вручную |

**Метрики:** не применимо (поведение, не продукт).

---

## 2. Созвоны и голосовые

**Суть:** Позвонить или отправить голосовое: «давай на этой неделе встретимся, когда тебе удобно?» — обсуждение в реальном времени.

| Плюсы | Минусы |
|-------|--------|
| Быстро договориться один на один | Неудобно для группы: не соберёшь всех на линию |
| Сразу слышно интонацию, можно уточнить | Нужно «ловить» момент, когда человек свободен ответить |
| | Не остаётся письменной фиксации времени и места |

**Метрики:** не применимо.

---

## 3. Опросы в мессенджерах (Telegram, WhatsApp)

**Суть:** Создать опрос в чате: «Когда встречаемся?» — варианты «Суббота 12:00», «Суббота 18:00», «Воскресенье» и т.д.

| Плюсы | Минусы |
|-------|--------|
| Всё в привычном чате, не нужно переходить по ссылкам | Ограниченное число вариантов, неудобно для многих слотов |
| Видят все участники чата | Нет привязки к календарю, нет напоминаний |
| Не нужна регистрация | Сложно комбинировать «дата + время» и «место» |
| | Нет автоматической финальной «конкретики» (где, когда, во сколько) |

**Метрики:** не применимо (функция мессенджера).

---

## 4. Doodle — опросы на время

**Суть:** Сервис для выбора времени встречи: организатор создаёт опрос с датами и временем, участники отмечают удобные слоты, Doodle показывает пересечения.

**Сайт:** https://doodle.com  
**Групповые опросы:** https://doodle.com/en/product/polls/

| Плюсы | Минусы |
|-------|--------|
| Классика жанра, узнаваемый бренд | Много рекламы в бесплатной версии, отвлекает |
| Участникам не нужен аккаунт | Ориентация на бизнес/офис; для друзей может казаться избыточным |
| Интеграция с календарём, видеозвонками (Zoom, Meet и т.д.) | Отдельный сайт — нужно уходить из чата |
| Поддержка часовых поясов | |

**Метрики (публичные данные):**  
- Около **133 млн пользователей**, сервис существует более 18 лет.  
- Опрос 2500 пользователей (State of Meetings 2023): 43% встреч в Северной Америке — на 6–10 человек; более 55% планируются за 1–4 недели.  
- Оценка рынка: глобальный рынок ПО для планирования встреч рос на сотни миллионов долларов до 2025 года.

## 5. Calendly — бронирование слотов

**Суть:** Организатор настраивает «страницу доступности» (когда готов встречаться); гости выбирают слот и записываются. Один на один или групповые события (вебинары, интервью).

**Сайт:** https://calendly.com

| Плюсы | Минусы |
|-------|--------|
| Удобно для повторяющихся типов встреч (звонки, интервью) | Заточен под «один организатор — гости», не под равноправный выбор времени группой друзей |
| Интеграции: календарь, Zoom, Salesforce, Slack и др. | Платный функционал для продвинутых сценариев |
| Автоматические напоминания и отмена | Участникам нужно переходить на сайт/страницу |
| | Избыточен для неформальных встреч друзей |

**Метрики (публичные данные):**  
- **10+ млн пользователей** (2022), **2,1+ млн платящих**.  
- **100 000+ организаций**, 86% компаний из Fortune 500 используют.  
- Выручка: ~70 млн $ (2020), ~100 млн $ (2021), ~144 млн $ (2022).  
- Раунд финансирования (2021): **350 млн $**, оценка **~3 млрд $**.  
- Доля рынка сервисов планирования — порядка 21% (2022).

## 6. When2meet — простой опрос без регистрации

**Суть:** Бесплатный веб-инструмент: создаётся событие с диапазоном дат и времени, участники по ссылке отмечают доступность в сетке (шаг 15 минут). В реальном времени видно, когда все свободны.

**Сайт:** https://www.when2meet.com (есть также when2meet.co.uk и др.)

| Плюсы | Минусы |
|-------|--------|
| Без регистрации и логина | Нет интеграции с календарём и мессенджерами |
| Наглядная сетка «кто когда свободен» | Отдельный сайт — уход из чата |
| Обновление в реальном времени | Минималистичный интерфейс, нет бренда и поддержки как у крупных игроков |
| Бесплатно, без навязчивой рекламы | Нет автоматических напоминаний и финального «где, когда, во сколько» |

**Метрики:** публичных данных о пользователях и выручке нет; популярен в студенческой и open-source среде.

---

## 7. WhenIsGood — опрос «когда удобно»

**Суть:** Организатор отмечает свои удобные слоты и отправляет ссылку; участники отмечают свои; только организатор видит полную картину (приватность ответов).

**Сайт:** https://whenisgood.net

| Плюсы | Минусы |
|-------|--------|
| Без регистрации | Отдельный сайт, нет связи с чатом |
| Приватность: отдельные ответы видит только создатель опроса | Нет календарной интеграции и автоматических уведомлений |
| Простой интерфейс | Мало известен массовой аудитории |

**Метрики:** не публикуются; небольшой независимый проект.

---

## 8. Google Calendar / Outlook — общий календарь и предложение времени

**Суть:** Просмотр общего календаря (если есть доступ) или отправка «предложения времени» (Meeting Poll в Outlook, предложение слотов в Gmail/Calendar).

| Плюсы | Минусы |
|-------|--------|
| Уже используются в работе и частично в личной жизни | Друзья редко синхронизируют личные календари друг с другом |
| Видно занятость, можно предложить слоты | Ассоциация с работой; для дружеских встреч часто избыточно |
| Интеграция с почтой и встречами | Нужны аккаунты и настройка прав доступа |

**Метрики:** Google Calendar — сотни миллионов пользователей (часть экосистемы Google); точные цифры по сценарию «друзья выбирают время» не выделяются.

---

## 9. Бумага / заметки (список, таблица)

**Суть:** Организатор пишет в чат или в общий документ (Google Таблица, Notion): «отметь, когда можешь» — участники вручную дописывают свои варианты.

| Плюсы | Минусы |
|-------|--------|
| Максимальная гибкость и контроль формата | Время на создание и поддержание таблицы/документа |
| Можно совместить время, место, идеи | Легко запутаться при большом числе участников и слотов |
| | Нет автоматического подсчёта пересечений и напоминаний |

**Метрики:** не применимо.

## Сводка по позиционированию Вайба

| Решение | Сильная сторона | Слабость для сценария «друзья в чате» |
|---------|-----------------|----------------------------------------|
| Переписка | Всё в чате | Долго, теряется, нет структуры |
| Опросы в Telegram/WhatsApp | В чате | Мало слотов, нет конкретики в конце |
| Doodle | Проверенный опрос на время | Реклама, отдельный сайт, «офисный» образ |
| Calendly | Автоматизация для организатора | Не про равный выбор группы друзей |
| When2meet / WhenIsGood | Просто, без регистрации | Отдельный сайт, нет связи с мессенджером |
| Календари | Видна занятость | Друзья редко делятся календарём |
| Вайб (целевой продукт) | В чате (Telegram), быстрый выбор слотов, конкретика в конце, без перехода на сторонние сайты | Пока нет готового продукта |
