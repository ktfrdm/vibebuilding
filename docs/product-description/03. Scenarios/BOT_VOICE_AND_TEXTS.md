# Тон общения и тексты бота

Инструкции по формулировкам сообщений бота. Используются в [BOT_FLOW_MVP](BOT_FLOW_MVP.md).

---

## Тон общения

**Общий стиль:** дружелюбный, тёплый, ориентированный на ламповую встречу друзей. Без панибратства и эмодзи.

**Для участников:**
- Без морального давления — отказ принять нормален
- Радость от предстоящей встречи, но без навязчивости
- Душевность в формулировках

**Для организатора:**
- Простота и скорость решения задачи
- Минимум шагов, понятные подсказки

---

## Тексты бота (для копирования)

| Контекст | Текст |
|----------|-------|
| Старт | «Давай соберёмся!» / «Собрать друзей» |
| Название | «Как назовём нашу встречу? Например: «Кофе», «Ужин в пятницу». Можно пропустить!» |
| Период | «Когда хочешь встретиться? Напиши как удобнее: «в эту субботу», «25 февраля», «следующие выходные» — поймём» |
| Период, если не распознано | «Хм, не совсем понял. Напиши попроще, например: «суббота 8 марта»» |
| Слоты | «Какие варианты времени предложить участникам? Напиши, например: «суббота 12:00, 15:00, 18:00» или «в субботу с 12 до 18». Лучше 3–7 вариантов» |
| Подтверждение слотов | «Отлично! Вот что получается: [список]. Подходит?» |
| Создано | «Готово! Поделись ссылкой с участниками — пусть выберут удобное время: [link]» |
| Приглашение в чат | «[Название] — когда вам удобно?» |
| Режим ответа участника | «Отметь удобные слоты или нажми «Увы, не смогу», если не получится» |
| При отказе («не смогу») | «Спасибо, что ответил!» |
| Участник (выбор времени) | «Выбери удобные тебе варианты» |
| Подтверждение участнику | «Супер, записал! Дождёмся, когда все ответят — тогда напишем, когда встречаемся» |
| Место | «Где собираемся? Напиши место или нажми «Пропустить» — уточните потом в чате» |
| Финальное уведомление (свободным) | «Встречаемся! [Название] — [дата, время]. Место: [место]» |
| Вопрос занятому | «[Название] назначена на [дата, время]. Сможешь прийти?» — кнопки «Да, приду!» / «Увы, не смогу» |
| При отказе занятого | «Жаль, надеемся увидеться в следующий раз!» |
