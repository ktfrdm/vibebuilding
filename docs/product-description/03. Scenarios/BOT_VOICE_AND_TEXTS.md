# Тон общения и тексты бота

Инструкции по формулировкам сообщений бота. Используются в [BOT_FLOW_MVP](BOT_FLOW_MVP.md).

---

## Тон общения

**Общий стиль:** дружелюбный, тёплый, ориентированный на ламповую встречу друзей. Без панибратства.

**Эмодзи:** не больше **одного** на пользовательское сообщение (акцент внимания); не обязательно в каждом сообщении. На **кнопках** — по одному эмодзи, по смыслу действия (см. `bot/keyboards/inline.py`). Исключение: слоты участника (✅/☐) — наглядность выбора.

**Для участников:**
- Без морального давления — отказ принять нормален
- Радость от предстоящей встречи, но без навязчивости
- Душевность в формулировках

**Для организатора:**
- Простота и скорость решения задачи
- Минимум шагов, понятные подсказки

## Тексты бота (для копирования)

| Контекст | Текст |
|----------|-------|
| Старт | Кнопки: «🤝 Давай соберёмся!», «📋 Статус». Текстом по-прежнему можно «Давай соберёмся!» / «Собрать друзей» |
| Название | «Как назовём нашу встречу? Например: «Кофе», «Ужин в пятницу». Можно пропустить!» |
| Доп. блок в **группе** к шагам с вводом текста | Короткая подсказка: в группе отвечать на это сообщение бота своим текстом из‑за групповой приватности (см. `bot/chat_context.py`, `GROUP_ORGANIZER_REPLY_HINT`) |
| Опции для встречи | «Какие опции для встречи предложить участникам? Напиши, например: «суббота 12:00, 15:00, 18:00» или «в эту субботу», «25 февраля вечером» — поймём» |
| Опции, если не распознано | «Хм, не совсем понял. Напиши попроще, например: «суббота 12:00, 15:00, 18:00»» |
| Подтверждение слотов | «Отлично! Вот что получается: [список]. Подходит?» |
| Создано (личка) | Заголовок: встреча готова, осталось позвать людей; инструкция к «Переслать приглашение» (чат/собеседник, ссылка → сразу выбор времени); строка со ссылкой; пояснение к «Ответить на приглашение» — пройти тот же путь, что и участник. См. `slots_confirmed` в `organizer.py` |
| Создано (группа) | «Готово! …» + пост приглашения в чат (как в коде) |
| Приглашение в чат | «[Название] — когда вам удобно?» |
| Режим ответа участника | «Отметь удобные слоты или нажми «Увы, не смогу», если не получится» |
| При отказе («не смогу») | «Спасибо, что ответил!» |
| Участник (выбор времени) | «Выбери удобные тебе варианты» |
| Подтверждение участнику | «Супер, записал! Дождёмся, когда все ответят — тогда напишем, когда встречаемся» |
| Место | «Где собираемся? Напиши место или нажми «Пропустить» — уточните потом в чате» |
| Финальное уведомление (свободным) | «Встречаемся! [Название] — [дата, время]. Место: [место]» |
| Вопрос занятому | «[Название] назначена на [дата, время]. Сможешь прийти?» — кнопки «Да, приду!» / «Увы, не смогу» |
| При отказе занятого | «Жаль, надеемся увидеться в следующий раз!» |
