## Примеры

### Пример 1: Регистрация
**Как** новый пользователь,  
**Я хочу** зарегистрироваться в боте через команду /start,  
**Чтобы** получить доступ к основным функциям бота.

**Критерии приемки:**
- [ ] Пользователь может отправить команду /start
- [ ] Бот приветствует пользователя и объясняет возможности
- [ ] Пользователь видит главное меню с доступными функциями

**Приоритет:** Высокий

---

### Пример 2: Получение помощи
**Как** пользователь,  
**Я хочу** получить список доступных команд через /help,  
**Чтобы** понять, как использовать бота.

**Критерии приемки:**
- [ ] Команда /help доступна в любой момент
- [ ] Бот показывает список всех доступных команд
- [ ] Описание команд понятное и краткое

**Приоритет:** Высокий
