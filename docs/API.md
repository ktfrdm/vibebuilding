## Команды бота

### /start
**Описание:** Команда запуска бота. Приветствует пользователя и показывает главное меню.

**Использование:** `/start`

**Ответ:** Приветственное сообщение с описанием возможностей бота.

**Пример:**
```
Пользователь: /start
Бот: Привет, Иван! 👋

Добро пожаловать в бота!
```

---

### /help
**Описание:** Показывает список доступных команд и их описание.

**Использование:** `/help`

**Ответ:** Список команд с описанием.

**Пример:**
```
Пользователь: /help
Бот: 📖 Доступные команды:

/start - Начать работу с ботом
/help - Показать это сообщение
```

## Типы сообщений

### Текстовые сообщения
**Описание:** Обработка обычных текстовых сообщений от пользователя.

**Текущее поведение:** Эхо-ответ (отправляет обратно текст пользователя)

**Пример:**
```
Пользователь: Привет!
Бот: Вы написали: Привет!
```

## Обработка ошибок

### Ошибка конфигурации
**Описание:** Бот не может запуститься из-за отсутствия токена.

**Решение:** Проверьте файл `.env` и убедитесь, что `BOT_TOKEN` установлен.
